# Endatix API Docs

Welcome to the Endatix API documentation!

## What it is

A robust, .NET-based backend designed for high-scale data collection. It handles everything from schema validation to secure data persistence.

## Tech Stack

- **.NET 10** — Modern .NET runtime with high performance
- **Entity Framework Core** — Powerful ORM for database operations
- **PostgreSQL/SQL Server** — Enterprise-grade database support

## Licensing

**Open Source (MIT)** — Free for commercial use.

## Key Capabilities

- **Headless form management** — Complete API-driven form lifecycle management
- **Extensible plugin architecture** — Custom logic and integrations through plugins
- **Enterprise-grade security and authentication** — Built-in JWT, OAuth, and Keycloak support
