# Endatix API (.NET)

Endatix API is the open-source .NET backend for SurveyJS. It provides persistence, REST endpoints, and event handling for form-centric applications.

## What you get

- **Fluent API** — Chained configuration for initialization
- **Forms & submissions** — Store JSON form schema and submissions in your database
- **REST API** — CRUD endpoints for any frontend (React, Vue, Angular, etc.)
- **Event handling** — Subscribe to form submission events and run workflows
