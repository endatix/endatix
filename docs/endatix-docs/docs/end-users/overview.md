# Hub Basics

Endatix Hub is a powerful form management platform that lets you create, customize, and manage forms without writing any code. Built on top of SurveyJS, it provides everything you need to build professional form solutions.

## What You Can Do

### Build Any Form

- Create forms with 20+ question types (text, checkboxes, uploads, matrix, etc.)
- Use conditional logic to show/hide questions based on responses
- Support multiple languages for global audiences
- Design mobile-ready forms that work on any device

### Customize Your Platform

- White-label the platform with your brand colors and logo
- Create custom question types with drag-and-drop editor
- Configure themes and styling to match your brand
- Set up custom domains for seamless integration

### Manage Submissions

- View and export form submissions
- Create custom status types to track actionable items
- Set up automated workflows with webhooks
- Integrate with your CRM, email marketing, and other tools

### Ensure Data Privacy

- HIPAA, GDPR, and CCPA compliant
- Role-based access controls
- Detailed audit trails
- Encryption at rest and in transit

## Platform Overview

Endatix Hub is designed to be the complete solution for form-centric applications, offering:

- **Advanced Form Builder**: Create complex forms with 20+ question types
- **Custom Question Types**: Build reusable components with drag-and-drop editor
- **Form Management**: Centralized backend for organizing forms and submissions
- **White-labeling**: Full control over appearance and branding
- **Integrations**: Connect with CRM, email marketing, and custom systems
- **Data Privacy**: HIPAA, GDPR, and CCPA compliant

## Quick Start

1. **Access your platform** — Log in to your Endatix Hub instance.
2. **Create your first form** — Click "Create New Form" and add questions using the form builder.
3. **Publish and share** — Test, publish, and share the form URL or embed it on your website.
4. **Monitor and manage** — View submissions in the dashboard and export or integrate data.
